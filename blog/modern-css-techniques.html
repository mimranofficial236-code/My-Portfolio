Modern CSS Techniques You Should Know

Muhammad Imran December 28, 2024 10 min read

CSS has evolved dramatically. These modern techniques will help you write cleaner, more maintainable stylesheets.

1. Container Queries

Finally, we can style elements based on their container size, not just the viewport:

.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: flex;
    gap: 1rem;
  }
}

2. CSS Nesting

Native CSS nesting is hereâ€”no preprocessor needed:

.card {
  padding: 1rem;
  
  & .title {
    font-size: 1.5rem;
  }
  
  &:hover {
    transform: translateY(-5px);
  }
}

3. :has() Selector

The "parent selector" we've always wanted:

/* Style form groups that contain invalid inputs */
.form-group:has(input:invalid) {
  border-color: red;
}

Pro Tip: Check browser support on caniuse.com before using these features in production.

Conclusion

Modern CSS is more powerful than ever. Embrace these features to write better, more maintainable code.
